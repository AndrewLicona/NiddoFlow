# NiddoFlow

NiddoFlow is an offline-first family financial management application.

## Structure

- `/frontend`: Next.js application (App Router, TypeScript, Tailwind CSS)
- `/backend`: FastAPI application (Python)

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
